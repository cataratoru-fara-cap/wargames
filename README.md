# Wargames

This repository is a local collection of challenge solutions and assets for several classic security wargame series.

## Structure

- `bandit/`
  - Contains the OverTheWire Bandit wargame materials.
  - Includes challenge passwords in `passwords/`, supporting resources in `resources/`, and solution notes or scripts in `solutions/`.

- `leviathan/`
  - Contains the OverTheWire Leviathan wargame materials.
  - Includes challenge passwords in `passwords/`, supporting resources in `resources/`, and solutions in `solutions/`.

- `narnia/`
  - Contains the OverTheWire Narnia wargame materials.
  - Includes challenge passwords in `passwords/`, resources in `resources/`, and solutions in `solutions/`.

- `natas/`
  - Contains the OverTheWire Natas wargame materials.
  - Includes challenge passwords in `passwords/`, payloads and helper scripts in `resources/`, and solutions in `solutions/`.

## How to use this repository

1. Choose the wargame folder you want to inspect: `bandit`, `leviathan`, `narnia`, or `natas`.
2. Review password files in the `passwords/` directory to see stored credentials for each level.
3. Inspect `resources/` for challenge binaries, helper scripts, and other artifacts needed to solve levels.
4. Read `solutions/` for notes, commands, or writeups describing how each level was solved.

## Notes

- This repository is intended as a reference for solving or reviewing OverTheWire challenge levels.
- Some directories may contain additional nested resources and README files specific to particular levels.
- Use the included files as a study aid, not as a replacement for practicing challenges on the game servers.
